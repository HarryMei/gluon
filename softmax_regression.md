## 前言

前面实现线性回归的简单实例，这里开始softmax回归的部分，该模型是logistic回归模型在多分类问题上的推广，而logistic回归以看成是线性回归的推广，所以从线性回归到softmax回归还是有很好的连贯性的，另外一个回归一个分类，也很好的代表了机器学习的中的两大类问题。

参照gluon官方提供的向导文档，我们也换掉老的MNIST手写图像数据集，而改用新的FashionMNIST服饰图像数据集。我们将使用softmax回归将建立一个能够很好区分FashionMNIST数据集图像类别的多分类器。

## 获取FashionMNIST数据集

使用gluon中的data.vision.FashionMNIST类获取数据，一下代码为向导文档中提供（自动下载、导入、转化数据集）：

```{.python .input  n=2}
from mxnet import gluon
from mxnet import autograd as ag
from mxnet import ndarray as nd
import numpy as np
import time

def transform(data, label):
    return data.astype('float32')/255, label.astype('float32')
mnist_train = gluon.data.vision.FashionMNIST(train=True, transform=transform)
mnist_test = gluon.data.vision.FashionMNIST(train=False, transform=transform)
```

## 定义模型

用softmax函数结合线性函数来定义分类模型

```{.python .input  n=3}
def softmax(x):
    exp = nd.exp(x).T
    partition = exp.sum(axis=1, keepdims=True)
    return exp/partition

def linear_module(x, K):
    return nd.dot(K, x.T)

def net(x, K):
    return softmax(linear_module(x, K))
```

softmax函数一般用于多分类问题上，为直观查看softmax的作用，我设计一个二分类问题：

假设一个事件发生有两种可能的情况：**情况1**，在图像中用红线表示；**类型2**，在图像中用绿线表示。一组数据值有两个数据，分别表示类型1的发生的次数和类型2发生的次数。当输入的两种类型发生的次数变化时，经过分别经过线性拟合的结果，然后再经过softmax最终输出的变化情况。

运行以下代码，可以看到softmax函数有如下功能：

+ 可以将输入的数据映射到[0, 1]之间
+ 是单调函数
+ 可以实现易于分类的S型曲线，将出现次数多的类型的概率提高，将出现次数少的类型概率减小

```{.python .input  n=4}
import numpy as np

data = nd.ones((20, 2))
data[:,0] = nd.arange(-5,15)
param = nd.array([1,1,2,1]).reshape((2,2))
# print(param)
plt.plot(data[:,0].asnumpy(), linear_module(data, param)[0].asnumpy(), '-*g')
plt.plot(data[:,0].asnumpy(), linear_module(data, param)[1].asnumpy(), '-.r')
plt.show()
# print(net(data, param))
plt.plot(data[:,0].asnumpy(), (net(data, param).T)[0].asnumpy(), '-*g')
plt.plot(data[:,0].asnumpy(), (net(data, param).T)[1].asnumpy(), '-.r')
plt.show()
```

## 定义代价函数

我们知道在logistic回归的参数估计时，由于其方差参数函数不是凸函数，不好优化，所以使用了最大似然法来估计来构造代价函数，同理可以推广到softmax回归。

这里我使用交叉熵来构造代价函数表示预测和实际的差距。由于多类分类将标签向量化后，肯定只有一个元素是1，所以依照公式：`yvec[0]*log(yhat[0])+...+yvec[n]*log(yhat[n])`只需一项是有效的，所以为了简化计算可以只计算那一项,使用pick函数提取有用的那一项。

```{.python .input  n=5}
def cost_fuction(yhat, y):
    return - nd.sum(nd.log(nd.pick(yhat, y)), axis=0)
```

```{.python .input  n=6}
def grad_descent(params, rate):
```

```{.python .input  n=7}
def accuracy(output, label):
    return nd.mean(output.argmax(axis=1)==label).asscalar()

def evaluate_accuracy(data_iterator, net, params):
    acc = 0.
    param_num = params.shape[1]
    for data, label in data_iterator:
        data_in = nd.ones((data.shape[0],param_num))
        data_in[:,:-1] = data.reshape((data.shape[0],param_num-1))
        output = net(data_in, params)
        acc += accuracy(output, label)
    return acc / len(data_iterator)
```

```{.python .input  n=8}
def softmax_learning(data_iter, params, iter_num, learning_rate):
    params.attach_grad()
    param_num = params.shape[1]
    acc_list = []
    loss_list = []
    for epoch in range(iter_num):
        train_loss = 0.
        train_acc = 0.
        for data, label in data_iter:
            data_in = nd.ones((data.shape[0],param_num))
            data_in[:,:-1] = data.reshape((data.shape[0],param_num-1))
            with ag.record():
                yhat = net(data_in, params)
                loss = cost_fuction(yhat, label)
            loss.backward()
            grad_descent(params, learning_rate/(epoch+1))
            
            train_loss += nd.mean(loss).asscalar()
            train_acc += accuracy(yhat, label)
        acc = train_acc/len(data_iter)
        loss_it = train_loss/len(data_iter)
        acc_list.append(acc)
        loss_list.append(loss_it)
        print('Epoch %d. Loss: %f, Train acc %f'%(epoch, loss_it, acc))
    return acc_list, loss_list
```

## 使用gluon的模型

使用gluon中的Flatten层将输入的图片数据集中图片降为1维向量(整个数据集将为2维向量)，然后建立输入到10个输出节点的全连接层。

在gluon中softmax和交叉熵是一起设定的，而网络中没有加入softmax有点奇怪，不过这个在训练的时候是一样的，因为代价函数时进行了softmax的。其他部分代码和手动写的一样。

```{.python .input  n=33}
gluon_net = gluon.nn.Sequential()
with gluon_net.name_scope():
    gluon_net.add(gluon.nn.Flatten())
    gluon_net.add(gluon.nn.Dense(10))
gluon_net.initialize()

softmax_cross_entropy = gluon.loss.SoftmaxCrossEntropyLoss()
trainer = gluon.Trainer(gluon_net.collect_params(), 'sgd', {'learning_rate':0.3})

def gluon_softmax_learning(data_iter, iter_num):
    acc_list = []
    loss_list = []
    for epoch in range(iter_num):
        train_loss = 0.
        train_acc = 0.
        for data, label in data_iter:
            with ag.record():
                yhat = gluon_net(data)
                loss = softmax_cross_entropy(yhat, label)
            loss.backward()
            trainer.step(data.shape[0])
            
            train_loss += nd.mean(loss).asscalar()
            train_acc += accuracy(yhat, label)
        acc = train_acc/len(data_iter)
        loss_it = train_loss/len(data_iter)
        acc_list.append(acc)
        loss_list.append(loss_it)
        print('Epoch %d. Loss: %f, Train acc %f'%(epoch, loss_it, acc))
    return acc_list, loss_list

def gluon_evaluate_accuracy(data_iterator, net):
    acc = 0.
    for data, label in data_iterator:
        output = net(data)
        acc += accuracy(output, label)
    return acc / len(data_iterator)
```
